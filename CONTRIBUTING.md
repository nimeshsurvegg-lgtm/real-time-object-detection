# Contributing to Object Detection Assistant

Thank you for your interest in improving the Object Detection Assistant!

## How to Contribute

### 1. Reporting Bugs
If you encounter a crash or an issue (e.g., camera not releasing, memory leaks), please open a GitHub Issue. Include your OS, Python version, and a traceback if possible.

### 2. Submitting Pull Requests
1. **Fork** the repository.
2. **Clone** it locally.
3. **Create a branch** for your feature (`git checkout -b feature/NewFeature`).
4. **Make your changes**. 
5. **Test** to ensure the GUI remains responsive and the camera handles cleanup properly upon quitting.
6. **Commit** your changes (`git commit -m "Add NewFeature"`).
7. **Push** to your fork (`git push origin feature/NewFeature`).
8. **Open a Pull Request** against the main branch.
